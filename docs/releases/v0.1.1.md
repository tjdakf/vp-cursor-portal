# vp-cursor-portal v0.1.1

Patch release for two review findings found after the initial MVP release.

## Assets

- `vp-cursor-portal-setup.exe`: Windows installer that installs under `Program Files`.
- `vp-cursor-portal-win-x64.zip`: self-contained portable Windows x64 build.

## Fixes

- Keeps ratio-mapped portal target coordinates inside the target monitor bounds, including ratios near the bottom or right edge.
- Preserves loaded safety settings when rebuilding and saving configuration, instead of resetting them to defaults.

## Validation

- GitHub Actions Windows Build passed for PR #3 before merge.
- Added regression coverage for near-end portal ratio mapping.
- Added regression coverage for preserving loaded `SafetySettings`.

## Install Notes

For most users, download and run `vp-cursor-portal-setup.exe`.

Use the ZIP artifact only when you want a portable folder without installer registration.

The installer and executable are not code-signed yet, so Microsoft Defender SmartScreen may show an unknown publisher warning.
